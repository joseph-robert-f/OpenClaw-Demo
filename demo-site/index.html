Workflow snapshot

How this deploy moves from Slack to Shopify.

Orchestrator Slack intake

Summarizes request, sets Linear tasks, and pings Growth + Web Ops.

  • Task: “Draft plan” → Growth Agent
  • Task: “Publish hero + PDP” → Web Ops
Growth Agent Research & copy

Sources spring EDC trends, generates copy pack, SEO, hooks.

  • Uploads plan to Linear issue
  • Flags assumptions (animation intensity, CTA color)
Web Ops Publish

Updates hero, PDP, alt text, and verifies on desktop + mobile.

  • Checklist: copy fields, SEO, images, mobile QA
  • Deploy note posted back to Slack with URL
