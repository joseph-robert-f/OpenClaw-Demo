How the agents work together

Single Slack channel → orchestrated execution.

1) Orchestrator

Intake → plan → delegate

  • Clarifies goal, constraints, and definition of done
  • Creates the task breakdown (Linear)
  • Spawns Growth + Web Ops as needed
2) Growth Agent

Research → positioning → copy

  • Trend brief + key insights
  • Headline/value prop variants
  • SEO meta + keyword cluster + tags
3) Web Ops

Publish → QA → confirm live

  • Updates CMS fields (Shopify)
  • Checks mobile/desktop rendering
  • Reports a deploy note with URL + verifications
